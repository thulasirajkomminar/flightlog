# Using Flightlog

Once you're signed in, Flightlog is built around three pages that show up in the sidebar:

| Section                               | What it's for                                                               |
| ------------------------------------- | --------------------------------------------------------------------------- |
| [**Dashboard**](dashboard.md)         | The map and the totals — your "passport" view of where you've been.         |
| [**Search & Add**](search-and-add.md) | Look up a flight by number and date, then save it to your log.              |
| [**My Flights**](my-flights.md)       | The full list, grouped by year. Expand, delete, import or export from here. |

If you've never used it before, the natural order is roughly:

1. [Create your account](accounts.md) — register, sign in, done.
2. [Add a flight or two](search-and-add.md) so the dashboard has something to show.
3. Or, if you're coming from another tracker, [import a CSV](import-export.md).
4. Look at [the dashboard](dashboard.md) and feel smug.

Each page in this section walks through one screen at a time, with the screenshot of what you should be seeing.
